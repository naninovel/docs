Why Naninovel?

The storytelling engine developers trust

The #1 storytelling tool for the Unity game engine, Naninovel is a comprehensive framework for crafting interactive fiction. Refined over a decade and supported by a thriving community, it's trusted by thousands of developers to ship commercial titles across every platform.

✒️

Writer-Friendly Scripting

Author stories in text documents using NaniScript or with the visual Story Editor — no programming required. Branching, variables, and conditions are all built in.

📦

Ready Out of the Box

Ships with a complete set of UI, save/load, state rollback, localization, auto-voicing, dialogue triggers, and dozens of other systems — fully functional from day one.

🛠️

Endlessly Customizable

Every built-in feature can be extended, overridden, or replaced via a simple prefab tweak or a clean C# API, thanks to a carefully designed modular architecture.

🕹️

Works With Any Game

Use Naninovel as a visual novel template or enable Dialogue Mode and drop it into an existing project to add branching dialogues and cutscenes to a game of any genre.

✨

Professional Services

Dedicated support, GitHub-based installation, release streams, and advanced samples — your license includes 1 year of the Support Plan and permanent access to all future releases.

🏆

Battle Tested

Developed for over a decade, Naninovel has powered numerous hits across every platform — including mobile and console — and has become the industry standard for commercial visual novels.
