# 介绍

Naninovel 是 [Unity 游戏引擎](https://unity3d.com) 的扩展，具有一套供剧本作家使用的创作工具和供开发人员使用的 C# 框架，旨在简化叙事驱动内容的创建，从游戏内对话和过场动画到成熟的 [视觉小说](https://en.wikipedia.org/wiki/Visual_novel)。

![](https://www.youtube.com/watch?v=lRxIKDU9z4k)

::: info NOTE
虽然 Naninovel 不限制您使用 Unity 实现的目标，但某些内置功能有特定要求（支持的 Unity 版本、项目配置、目标平台）才能正常工作。有关更多信息，请参阅 [兼容性页面](/zh/guide/compatibility)。
:::

此在线文档是主要的官方参考，随引擎更改保持最新。剧本作家应从 [快速上手](/zh/guide/getting-started) 和 [Naninovel 脚本](/zh/guide/scenario-scripting) 指南开始，同时也鼓励开发人员阅读“开发”部分。“高级”部分侧重于 C# 框架，建议在将 Naninovel 集成到现有代码库或需要更深入的扩展和自定义时阅读。

如果您正在寻找特定主题，请尝试使用网站标题中的搜索栏，并查看 [常见问题](/zh/faq/) 以获取常见问题的答案。标准脚本命令和使用示例列在 [API 参考](/zh/api/) 中。如果找不到所需信息，请随时 [联系支持](/zh/support/#naninovel-support)。
