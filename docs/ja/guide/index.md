# 紹介

Naninovelは [Unityゲームエンジン](https://unity3d.com) の拡張機能であり、シナリオライター向けのオーサリングツールスイートと開発者向けのC#フレームワークを備えています。ゲーム内の会話やカットシーンから本格的な [ビジュアルノベル](https://en.wikipedia.org/wiki/Visual_novel) まで、物語主導のコンテンツ作成を効率化するように設計されています。

![](https://www.youtube.com/watch?v=lRxIKDU9z4k)

::: info NOTE
NaninovelはUnityで実現できることを制限しませんが、一部の組み込み機能が正しく動作するには特定の要件（サポートされているUnityバージョン、プロジェクト構成、ターゲットプラットフォーム）があります。詳細については、[互換性ページ](/ja/guide/compatibility) を参照してください。
:::

このオンラインドキュメントは主要な公式リファレンスであり、エンジンの変更に合わせて最新の状態に保たれています。シナリオライターは [スタートガイド](/ja/guide/getting-started) と [シナリオスクリプト](/ja/guide/scenario-scripting) ガイドから始めることをお勧めします。開発者は「開発者」セクションも読むことをお勧めします。「アドバンスド」セクションはC#フレームワークに焦点を当てており、Naninovelを既存のコードベースに統合する場合や、より深い拡張とカスタマイズが必要な場合に推奨されます。

特定のトピックを探している場合は、Webサイトヘッダーの検索バーを使用し、よくある質問への回答については [FAQ](/ja/faq/) を確認してください。標準のスクリプトコマンドと使用例は [APIリファレンス](/ja/api/) に記載されています。必要な情報が見つからない場合は、お気軽に [サポートにお問い合わせください](/ja/support/#naninovel-support)。
