Register Naninovel

Thank you for purchasing Naninovel! Registering will allow you to access direct support on the Naninovel discord server.

Fill in the invoice number and Discord ID fields in the form at the bottom of the page and press "Register" button to perform the registration.

You can find your invoice number (starts with "IN") in the PDF attached to the purchase confirmation email sent by Unity. If you have not received the invoice or can't find it, contact Unity support.

Your Discord ID will be revealed in the direct (private) message sent by MEE6 bot when you join the Discord server. In case you can't find the message or want to register another user, check the article on how to manually get the user ID. Make sure the user has joined Naninovel discord server before performing the registration!

When the registration is completed you will automatically receive a Verified User role on the server and access to the #support and #download channels.

If you have any issues during the registration, send a private message to @Elringus#6359 on Discord.
